## 🎯 About
Ravindra IAS is India's premier coaching institute for UPSC Civil Services Examination preparation. This website showcases our comprehensive courses, results, and allows aspirants to connect with our expert faculty.
## ✨ Features
### Core Features
- **Responsive Design** - Optimized for mobile, tablet, and desktop
- **Interactive UI** - Smooth animations and transitions
- **Course Catalog** - 16+ specialized UPSC courses
- **Results Showcase** - Toppers gallery with AIR rankings
- **FAQ Section** - Accordion-style frequently asked questions
- **Contact Form** - Free counseling request with validation
- **Newsletter Subscription** - Stay updated with latest news
### Premium Polish
- Page transition animations
- Hover zoom effects on images
- Animated counters
- Scroll progress indicator
- Back to top button
- Toast notifications
- Loading screen animation

## 🚀 Getting Started
### Prerequisites
- Node.js 18+
- npm or yarn
### Installation
```bash
# Clone the repository
git clone https://github.com/your-username/ravindra-ias.git
# Navigate to project directory
cd ravindra-ias
# Install dependencies
npm install
# Start development server
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.
### Build for Production
```bash
npm run build
npm start
```
